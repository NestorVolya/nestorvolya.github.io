# Profile

Олександр Корнієнко / «Нестор Воля» — дослідник і практик на перетині OSINT, інформаційних операцій, військового медіаконтексту та AI-assisted automation.

## Positioning

Ключова сильна сторона профілю — здатність поєднувати польовий збір, довгострокову роботу з мережами, доказову аналітику і технічну автоматизацію. Не просто знаходить факти, а збирає їх у відтворюваний evidence chain:

`source -> extraction -> verification -> contradiction handling -> judgment -> report`

## Tracks

| Track | Summary |
|---|---|
| OSINT investigations | корпоративний OSINT, attribution, source provenance, narrative analysis, public records, domains, web archives |
| InfoOps / StratCom | медіаприсутність, рекрутингові кампанії, робота зі ЗМІ, public event documentation |
| AI / automation | Python, Playwright, PostgreSQL/pgvector, LLM extraction, actor-profile pipelines |
| Network research | довгострокова робота з діаспорними, волонтерськими і публічними мережами |

## Working style

- Evidence before judgment.
- Raw sources separated from analysis and outputs.
- Contradictions treated as signals, not noise.
- Reports built for action: commander brief, recruiter CV, public publication, or internal decision flow.
- Automation used to increase repeatability, not to replace human judgment.

## Scope and safety

This public profile does not include private identifiers, live internal infrastructure, current sensitive operational details, or raw source dumps.
